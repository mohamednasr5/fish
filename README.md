# 🐟 تطبيق إدارة مزرعة الاستزراع السمكي

تطبيق ويب متكامل لإدارة مزارع الاستزراع السمكي مع دعم PWA للتثبيت على الأيفون.

## 🚀 رفع على GitHub Pages

### الخطوات:

1. **إنشاء Repository جديد على GitHub:**
   - اذهب إلى github.com وسجّل الدخول
   - اضغط على `+` ثم `New repository`
   - اسم المستودع: `fish-farm` (أو أي اسم)
   - اجعله **Public**
   - اضغط `Create repository`

2. **رفع الملفات:**
   - ارفع جميع الملفات: `index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`
   - يمكن سحبها وإفلاتها مباشرة في صفحة المستودع

3. **تفعيل GitHub Pages:**
   - اذهب إلى `Settings` في المستودع
   - في القائمة الجانبية اختر `Pages`
   - في `Source` اختر `Deploy from a branch`
   - اختر `main` أو `master` ثم `/ (root)`
   - اضغط `Save`

4. **الرابط سيكون:**
   ```
   https://[اسم-المستخدم].github.io/fish-farm/
   ```

## 📱 التثبيت على الأيفون (PWA)

1. افتح الرابط في **Safari** على الأيفون
2. اضغط على زر **المشاركة** ↑ في أسفل الشاشة
3. اختر **إضافة إلى الشاشة الرئيسية**
4. اضغط **إضافة** ✅
5. سيظهر التطبيق على الشاشة الرئيسية مثل أي تطبيق عادي!

## 🗄️ قاعدة البيانات

يستخدم التطبيق **Firebase Realtime Database** لتخزين البيانات بشكل تلقائي.

## 📋 مميزات التطبيق

- ✅ إدارة 5 قطاعات (الأول، الثاني، الثالث، الصرف، المغزى)
- ✅ إضافة أنواع الأسماك (بوري، طوبار، شبار، بياض) بالكيلو أو الطن
- ✅ إدارة المخزن (3 أنواع علف: 30%، 18%، 25% - عايم وغاطس)
- ✅ صرف علف لكل قطاع مع تتبع المتبقي
- ✅ سجل عمليات كامل قابل للفلترة
- ✅ لوحة تحكم شاملة مع إحصائيات
- ✅ تصميم متوافق 100% مع الأيفون
- ✅ دعم PWA - تثبيت على الشاشة الرئيسية
- ✅ يعمل بدون إنترنت (Offline) بعد التثبيت
